# Desafios Simples com Condicionais em Python

Este repositório reúne exercícios básicos em Python com foco no uso de estruturas condicionais, especialmente `if`, `elif` e `else`.

## Objetivo

Este projeto foi criado para:

- Revisar o uso das instruções `if`, `elif` e `else`
- Praticar a criação de regras condicionais
- Desenvolver a lógica de programação com situações do dia a dia
- Exercitar entrada, processamento e saída de dados em Python

## Arquivos do projeto

Os desafios incluídos neste repositório são:

- `acesso_escritorio.py` — Verifica se o acesso ao escritório é permitido com base no horário informado
- `aprova_emprestimo.py` — Analisa se um empréstimo pode ser aprovado de acordo com a renda e o valor da parcela
- `calcula_pedagio.py` — Calcula ou avalia o valor do pedágio conforme as condições definidas no problema
- `calcula_tempo_projeto.py` — Trabalha com tempo estimado de projeto e análise de condições
- `calculadora_imc.py` — Calcula o IMC e classifica o resultado
- `classificando_media.py` — Classifica a média do aluno com base na nota final
- `controle_orcamento.py` — Avalia o orçamento e identifica possíveis situações de controle financeiro
- `monitorando_vendas.py` — Analisa resultados de vendas a partir de condições definidas
- `temperatura_servidores.py` — Verifica a situação dos servidores de acordo com a temperatura
- `verifica_paridade.py` — Identifica se um número é par ou ímpar

## Conteúdos praticados

Durante a resolução desses exercícios, são trabalhados conceitos como:

- Estruturas condicionais
- Comparações numéricas
- Operadores relacionais
- Operadores lógicos
- Tomada de decisão no código
- Organização básica de programas em Python
